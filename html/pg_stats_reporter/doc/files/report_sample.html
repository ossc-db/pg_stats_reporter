  • total_rollbacks : total rollback transactions in this database cluster

How to see a Database Statistics table.

  • database : database name
  • MiB : size of database (MiB)
  • +MiB : increase size of database (MiB)
  • commit/s : committed transactions per second
  • rollback/s : rollback transactions per second
  • hit% : cache hit's rate (shared_buffers hit's rate)
  • gets/s : read blocks from disk and disk cache per second
  • reads/s : read blocks from disk per second
  • rows/s : number of rows from tables and indexes per second
    *The small value is displayed as zero by the truncation of the number in unit conversion

  • x axis : time
  • y axis : transactions per second (xact/s)
  • [database name] commit_tps : commited transactions of [database name] per second (xact/s)
